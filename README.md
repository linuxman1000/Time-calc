 Time-calc
Converts seconds to bigger units (years, months, weeks, days, hours, minutes)
  
<b>Usage:</b>
<code>
timecalc.py [[-s][--seconds] seconds]
            [[-m][--minutes] minutes]
            [[-u][--hours] hours]
            [[-d][--days] days]
            [[-w][--weeks] weeks]
            [[-o][--months] months]
            [[-y][--years] years]
</code>
